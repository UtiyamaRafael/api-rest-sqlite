# API-completa

API REST desenvolvida com Node.js e PostgreSQL, oferecendo um CRUD completo com suporte a filtros, ordenação e paginação. A aplicação conta com validações robustas, uso correto de status codes HTTP e base de dados previamente populada.

Como diferencial, possui autenticação via JWT e relacionamentos entre entidades (JOINs), seguindo boas práticas de desenvolvimento backend.
